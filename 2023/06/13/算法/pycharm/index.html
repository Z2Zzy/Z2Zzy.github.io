PyCharm的安装(Windows系统)

发表于2023-06-13|更新于2023-12-12|算法
|字数总计:366|阅读时长:1分钟|阅读量:

1.安装包下载

前往PyCharm官网下载安装包。
日常使用只需要安装Community版本就可以了。
点击Download Now下载。

2.安装

双击下载完成的.exe文件。

点击Next进入下一步。

选择安装的位置，由于PyCharm所需空间较多，建议不要安装在C盘。点击Next进入下一步。

数字1 : create desktop shortcut(创建桌面快捷方式），系统32位就选32-bit,系统64位就选64-bin。现在大多数都是64位了，不清楚系统类型:我的电脑=》右键-》属性。
数字2 ‘ update path variable(restart needed)更新路径变量(需要重新启动)，add lunchers dir to the patn(将启动器目录添加到路径中)。上一个PyCharm版本没有的，所以没有选择。
数字3 : update context menu(更新上下文菜单), add open folder as project(添加打开文件夹作为项目)。上一个PyCharm版本没有的，所以没有选择
数字4 : create associations 创建关联，关联.py文件，双击都是以pycharm打开。
数字5∶不要选.（数字5如果有需求，自己可以单独去官网下载安装JRE )
选择完成后点击Next进入下一步。

默认安装即可，点击Install。

等待安装完成。

点击Finish，安装完成。

3.汉化

打开PyCharm软件。

点击左上角File，选择Settings。

选择Plugins，搜索栏搜索Chinese，安装汉化包。

重启PyCharm软件即可。
